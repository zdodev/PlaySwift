###  Baekjoon 5338번 마이크로소프트 로고

단순히 다음과 같은 출력을 하는 것이 문제입니다.

```swift
     /~\
    ( oo|
    _\=/_
   /  _  \
  //|/.\|\\
 ||  \ /  ||
============
|          |
|          |
|          |
```

한 줄에 다 적어도 좋고 한 줄씩 print() 메서드를 사용해도 좋습니다. 다만 중간에 역슬래시`\` 경우 특수 기호이므로 역슬래시 `\` 를 출력하기 위해서는 역슬래시 `\` 역슬래시 `\` 두 번 적어야합니다.
